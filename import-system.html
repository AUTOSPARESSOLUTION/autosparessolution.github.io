Import System

Import customers or products from CSV files. Existing records will be updated (by email for customers, by SKU for products) or added if new.

Import Customers

Expected columns: email, name, phone, address, gstin, state, creditLimit, status (optional). First row header required.

Import Products

Expected columns: sku, name, brand, make, group, lineCode, price, currentStock, reorderLevel, hsn, unit. First row header required. Only sku and name are mandatory.
